Mi Fila

Mi Fila, es un software web, el cual está complementado con una app móvil de solicitud y segumiento de turnos. Esta herramienta tecnológica permite de forma sencilla y ordenada gestionar filas. Los usuarios desde un dispositivo móvil pueden generar y seguir los turnos; también lo pueden solicitar de forma presencial, asignados por asesor o a través de un dispensador(No incluímos equipos). Para los asesores es, de gran utilidad para realizar llamados de forma eficiente; el sistema por medio de una pantalla indica a el usuario el número de turno y el módulo donde recibira atención. Además de las características mencionadas anteriormente, el sistema también cuenta con un módulo de chat; el cual permite una comunicación efectiva entre los integrantes de la empresa.
